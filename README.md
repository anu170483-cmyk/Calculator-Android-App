# Android Calculator App

A simple Android calculator application built using **Java** in **Android Studio**.
The app performs basic arithmetic operations and provides a user-friendly calculator interface similar to a standard mobile calculator.

# Features

* Basic mathematical operations

  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
  * Modulus (%)

* Integer and floating-point calculations

* Continuous operations (e.g., `5 + 5 + 5`)

* Clear (AC) and backspace (C) functionality

* Prevents multiple decimal points in a number

* Handles division by zero safely

* Responsive UI that adapts to different screen sizes



# Technologies Used

* **Java**
* **Android Studio**
* **XML (for UI design)**
* **GridLayout / LinearLayout**
  

## How to Run the Project

1. Clone the repository


        git clone https://github.com/yourusername/calculator-app.git

3. Sync Gradle

4. Run the application on:

   * Android Emulator
   * Physical Android device
